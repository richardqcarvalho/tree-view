# Tree View

A data visualization with parents and children, organized as a tree with elements and sub-elements, search for name and filtering by status and type

## Technologies used

- TypeScript
- Vite
- React
- React Router
- TailwindCSS
- TanStack Query (React Query)
- Zustand
